Use the Open AI api to build a chatbot for Laravel.

## Development

1. Clone the repo

```
git clone [github https url]
```


2. Install packages

```
yarn install
```

3. Set up your `.env` file

- Copy `.env.example` into `.env`

4. Ingest Data

```
yarn ingest
```

5. Start the app

```
yarn dev
```
